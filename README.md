# **Mikrotik**

Mikrotik Virtual machine RouterOS, připojení přes síťový most na virtual machine skrze WinBox.





## Příkazy:

``` 
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
## Migrate příkazy:

``` 
flask db init
flask db migrate
flask db upgrade
```
## Tutoriál pro zprovoznění mikrotiku skrze VirtualBox:
